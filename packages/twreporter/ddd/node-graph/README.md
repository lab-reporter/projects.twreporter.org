# node-graph

A small Svelte app for browsing and editing graph data.

## Stack

- Frontend
  - Vite
  - Svelte 5 SPA with `sv-router`
  - `@xyflow/svelte` for node graphs
- Backend: Convex
- Auth: Clerk with `svelte-clerk`

## Commands

```bash
pnpm dev
pnpm convex:dev
pnpm check-types
pnpm build
```
